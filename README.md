# bs-chat-app

Chat app landing page

[Preview](https://indrasukma.github.io/bs-chat-app/)

## Installation

```
git clone https://github.com/IndraSukma/bs-chat-app.git
cd bs-chat-app
npm install
```

## Usage

### Start local development

```
npm start
```

### Build production files

```
npm run build
```
